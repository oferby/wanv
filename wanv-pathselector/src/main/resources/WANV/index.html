WAN Virtualization

The WAN Soft Switch is a software-only network element, running on a commodity x86 server, which provides a set of enterprise-grade WAN connectivity features on top of a set of low-cost WAN links. Such features include:

  • Link protection
  • Link aggregation
  • Flow-based Load-balancing
  • Flow-based quality of service
  • Pluggable WOC library
  • Pluggable DPI library (for OTT service flow classification)
  • Self-optimizing algorithm using Machine Learning
